NEM Wallet
===============

Hassle-free portable NEM Wallet for Linux, MacOS and Windows

## Start

```
git clone https://github.com/pocesar/node-nem-wallet.git
cd node-nem-wallet
npm install
npm run postinstall
npm run start
```
